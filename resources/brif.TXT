# Какой сайт мы создаём?
Онлайн школа по программированию 

# В чем заключается цель сайта?
- Онлайн обучение программированию
- Реклама популярных компаний(куда поступают выпускники онлайн школы)


# Кто целевая аудитория сайта?

1. Дети от 14 до 19 лет 
2. Взрослые в среднем 25 лет и больше

- Люди которые интерисуются программированием и которые не могут учиться очно (С любым образованием)

# Какие есть пожелания по внешнему виду сайта и пожелания по общению от использования сайта?

1. Минималистичный (без перегрузки цветами - белый, зелёные тона и цвета мягко сочетающиеся с зелёным)
2. В стиле IT (с изображениями связанными с программированием)
3. С изображениями профессионалов работающие в компании с каким-то описанием
4. С информацией о курсах:
    - название специальности
    - время обучения
    - краткое описание
    - цена
5. Форма записи на консультацию

# Какие либо особенности и пожелания
- форма регистрации пользователя
- онлайн курсы по программированию со специалистами
- возможность звонка с группой людей для создания урока
- админ панель сайта
- мессенджер с сотрудниками
- площадка для обучения программированию без преподавателя, задания :
    - Дополнение кода
    - Написание кода по ТЗ(на логику)
- мобильная версия(она же для разметки мобильного приложения)
- возможность проводить онлайн интенсивы для всех(не только учащихся) для привлечения новых учеников
- Подбор курса для людей которые не могут определиться
